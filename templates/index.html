Classic Defense

Code words fall from the top

                      bottom. Don't let them cross it.

                  

                

                

                  
2

                  

                    
Start typing to lock on & shoot

                    
Type the first letters of any word — the ship automatically aims at it.
                      Finish spelling the word and it's destroyed. No Enter needed.

                  

                

                

                  
3

                  

                    
One breach = game over

                    
If any word touches the red line, the mission fails immediately. Stay
                      sharp.

                  

                

              

              

                
+10 per word

                
+20 corrupt word

Code Builder

              

                

                  
1

                  

                    
Full lines of code fall instead of single words

                    
You'll see complete lines like for i in
                        range(3): falling toward the red line. These fall much slower than in Classic mode to
                      give you time to think and type.

                  

                

                

                  
2

                  

                    
Type the line exactly in the terminal below

                    
Type it character by character — spaces, brackets, quotes and all. The
                      system auto-targets the line you're typing. Finish it and it's destroyed.

                  

                

                

                  
3

                  

                    
PEEK to preview all lines before playing

                    
Press 👁 PEEK CODES (top-right) to pause the
                      game and see all lines for the current level. Study them, then resume.

                  

                

                

                  
4

                  

                    
An OUTPUT appears when you complete a line

                    
A small panel top-right shows you what that line of code actually
                      does — so you learn the syntax as you play.

Entry fee deducted when each level starts

                    
Level 1 costs −10 pts, Level 2 costs
                      −20 pts, and so on. You are only charged if you have enough points
                      to afford it. Complete the level to recover them — the +200
                      bonus more than covers it.
                    

                  

                

                

                  
6

                  

                    
Level 6 is the BOSS LEVEL

                      and loops together. A warning screen appears before it starts.

                  

                

              

              

                
+15 per line

1

                      misses and you lose a point.

Typos, wrong submissions — each one deducts 1 point in real time. Think
                      before you type.

Symbols & Operators used across all modes

            

              

                
=

                
Assignment

                
Stores a value into a variable. x = 5 means x
                  now
                  holds 5.

              

              

                
==

                
Equality Check

                
Compares two values. Returns true if they're equal. x
                    ==
                    5

              

              

                
!=

                
Not Equal

                
Returns true if values differ. x != 0

              

              

                
> <

                
Comparison

                
Greater-than / less-than. Used in conditions. if x
                    >
                    5

              

              

                
+

                
Add / Concat

                
Adds numbers or joins strings. "Hi " + name
                

              

              

                
- * /

                
Arithmetic

                
Subtract, multiply, divide. Standard math operators.

              

              

                
%

                
Modulo

                
Remainder after division. 10 % 3 = 1

              

              

                
( )

                
Parentheses

                
Groups expressions or passes arguments to functions. print("hi")

              

              

                
[ ]

                
Brackets

                
Defines a list or accesses an index. nums[0]
                

              

              

                
{ }

                
Braces

                
Code block (Java/C) or dictionary (Python). {'key':
                    val}

              

              

                
" "

                
String Literal

                
Text value. Everything between quotes is treated as plain text.

              

              

                
// # /* */

                
Comments

                
Notes in code the computer ignores. # this is a
                    comment

              

              

                
;

                
Statement End

                
Marks the end of a line in Java/C. Python doesn't need it.

              

              

                
:

                
Block Start

                
In Python, starts a block after if/for/def/class. if
                    x >
                    0:

              

              

                
→ indent

                
Indentation

                
In Python, 4 spaces inside a block means "this belongs to the if/for/def
                  above".
                

              

              

                
&& || !

                
Logic (Java/C)

                
AND, OR, NOT. Combine conditions. x > 0 &&
                    y >
                    0

              

              

                
and or not

                
Logic (Python)

                
Same as && || ! but written as words. x >
                    0 and y
                    > 0

              

              

                
+= -=

                
Compound Assign

                
Shorthand. x += 1 is the same as x = x + 1

              

            

          


          
          

            
Python — Keywords, Functions & Syntax

            

              

                
print()

                
Output

                
Displays text or values to the screen. print("Hello")
                

              

              

                
input()

                
User Input

                
Pauses and waits for the user to type something. Returns a string.

              

              

                
int()

                
Integer Cast

                
Converts a value to a whole number. int("42") →
                    42
                

              

              

                
str()

                
String Cast

                
Converts a value to text. str(42) → "42"

              

              

                
float()

                
Decimal Cast

                
Converts to decimal number. float("3.14") →
                    3.14

              

              

                
if / elif / else

                
Conditional

                
Runs code only if a condition is true. if x > 0:
                    ... else:
                    ...

              

              

                
for ... in

                
For Loop

                
Repeats code for each item in a sequence. for i in
                    range(5):

              

              

                
while

                
While Loop

                
Repeats as long as a condition stays true. while x
                    <
                    10:

              

              

                
def

                
Define Function

                
Creates a reusable block of code. def
                    greet(name):
                

              

              

                
return

                
Return Value

                
Sends a value back from a function. return x +
                    1

              

              

                
class

                
Class

                
Blueprint for creating objects. Groups data and behaviour together.

              

              

                
range()

                
Number Range

                
Generates a sequence. range(3) → 0, 1, 2

              

              

                
len()

                
Length

                
Returns how many items are in a list or how many characters in a string.
                

              

              

                
list / dict

                
Collections

                
list: ordered collection [1,2,3]. dict:
                  key-value
                  pairs {'a':1}

              

              

                
import

                
Import Module

                
Loads extra code/libraries. import math

              

              

                
try / except

                
Error Handling

                
Catches errors so the program doesn't crash. try: ...
                    except
                    ValueError:

              

              

                
lambda

                
Anonymous Fn

                
One-line inline function. f = lambda x: x * 2
                

              

              

                
global

                
Global Scope

                
Tells Python a variable lives outside the current function.

              

            

          


          
          

            
Java — Keywords, Types & Structure

            

              

                
int / double / char

                
Primitive Types

                
Declare the type of a variable. Must be specified before the name. int x = 5;

              

              

                
String

                
Text Type

                
Holds text. Capital S — it's an object, not a primitive. String name = "Alex";

              

              

                
boolean

                
True/False

                
Holds true or false. Used in conditions. boolean flag
                    =
                    true;

              

              

                
void

                
No Return

                
Method return type when nothing is returned. void
                    printHi()
                    {...}

              

              

                
public / private

                
Access Modifier

                
Controls who can access a method or variable. public = anyone, private =
                  only this
                  class.

              

              

                
static

                
Class-level

                
Belongs to the class, not an instance. public static
                    void
                    main

              

              

                
class

                
Class

                
Blueprint for objects. All Java code lives inside a class. public class Dog {...}

              

              

                
new

                
Create Object

                
Creates an instance of a class. Dog d = new
                    Dog();
                

              

              

                
this

                
Current Object

                
Refers to the current class instance. this.name =
                    name;

              

              

                
System.out.println

                
Print Output

                
Prints a line to the console. System.out.println("Hello");

              

              

                
if / else / switch

                
Control Flow

                
Conditional branching. switch tests one variable against multiple cases.
                

              

              

                
for / while

                
Loops

                
for(int i=0; i<5; i++) or while(x < 10)

              

              

                
import

                
Import

                
Brings in Java library classes. import
                    java.util.Scanner;

              

              

                
final

                
Constant

                
Value cannot be changed after assignment. final int
                    MAX =
                    100;

              

              

                
extends / implements

                
Inheritance

                
extends: inherit from a class. implements: fulfill an interface contract.
                

              

              

                
return

                
Return Value

                
Sends a value back from a method. Type must match declaration.

              

            

          


          
          

            
C-Lang — Directives, Types & Memory

            

              

                
#include

                
Header Include

                
Loads a library. #include <stdio.h>
                  gives access
                  to printf/scanf.

              

              

                
int / char / float

                
Primitive Types

                
C requires declaring the type before every variable. int age
                    = 20;

              

              

                
printf()

                
Print Output

                
Prints formatted text. printf("Hi %s\n",
                    name); — %s
                  inserts a string.

              

              

                
scanf()

                
Read Input

                
Reads user input. scanf("%d", &x); — &
                  passes
                  the memory address.

              

              

                
%d %s %f

                
Format Specifiers

                
Placeholders in printf/scanf. %d=int, %s=string, %f=float, %c=char.

              

              

                
\n \t

                
Escape Sequences

                
\n = new line. \t = tab. Used inside string literals.

              

              

                
* &

                
Pointer / Address

                
* declares/dereferences a pointer. & gets the memory address of a
                  variable.
                

              

              

                
malloc / free

                
Memory

                
malloc() allocates heap memory. free() releases it. Always pair them.
                

              

              

                
struct

                
Structure

                
Groups related variables under one name. C's version of an object.

              

              

                
sizeof()

                
Size Of

                
Returns byte size of a type or variable. Used with malloc.

              

              

                
int main()

                
Entry Point

                
Every C program starts here. Returns 0 to OS on success.

              

              

                
return 0

                
Exit Success

                
Tells the OS the program finished without errors.

              

              

                
if / else / switch

                
Conditionals

                
Same logic as other languages. Must use curly braces for blocks.

              

              

                
for / while / do-while

                
Loops

                
do-while runs at least once. do { ... }
                    while(cond);
                

              

              

                
void

                
No Return / Generic Ptr

                
Function returns nothing, or void* is a generic pointer type.

              

              

                
NULL

                
Null Pointer

                
A pointer that points to nothing. Always check for NULL after malloc.
