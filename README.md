# 🛡️ CodeDefender

A cyberpunk-themed typing game that teaches real programming concepts.
Type keywords to defend against incoming code threats across Python, Java, and C.

## Features
- Classic and Builder game modes
- Beginner-friendly Rookie difficulty
- In-game sandbox terminal (run real code)
- Global leaderboard and rank system
- Learning engine with concept tooltips

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: MySQL
